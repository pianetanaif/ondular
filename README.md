# Ondular Site Copy

This folder contains a local copy of `https://commonslab.ai/thesis` with its required static assets.

## Run locally

```bash
cd /Users/a10/ondular
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Edit content

- Main page: `index.html`
- Styles/scripts/fonts: `_next/static/...`

You can replace text and links directly in `index.html` with your own content.
